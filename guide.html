⏰ DAILY TRIGGER FREE 8:00 AM every day

n8n's built-in Schedule Trigger. Set to run at 8AM daily via cron: 0 8 * * *

Reads the state file to determine which chapter comes next, so it automatically advances each day without you doing anything.

📄 READ STATE LOCAL Tracks chapter progress

Reads /home/n8n/orv_state.json to see the last processed chapter. Increments by 1 for today's run.

{"lastChapter": 47}

This means each day the workflow picks up exactly where it left off, with no manual tracking.

🕷️ SCRAPE CHAPTER FREE HTTP Request + HTML Extract
⚠ Copyright Notice

Scraping manga sites without permission violates their ToS and copyright law. Your YouTube channel may get struck. Consider reaching out to scanlation groups for permission, or use officially free manhwa instead.

Sends an HTTP GET to the manga chapter page and extracts all <img> tags from the reader. Uses a browser User-Agent header to avoid bot detection.

GET https://manganato.com/manga-orv/chapter-48

Extracts image URLs matching the pattern used by the reader container.

⬇️ DOWNLOAD PANELS LOCAL wget via Execute Command

Downloads all panel images sequentially using wget. Saves them to:

/home/n8n/orv_chapters/chapter_048/
  page_001.jpg
  page_002.jpg
  page_003.jpg
  ...

Images are numbered with zero-padding so the renderer picks them up in correct order.

🤖 GEMINI AI SCRIPT API · FREE TIER gemini-1.5-flash

Calls Google Gemini 1.5 Flash (free tier: 15 RPM, 1M tokens/day) to generate a 950-word narration script for the chapter.

The prompt instructs Gemini to write in dramatic present tense, include character commentary, and structure it as a 7-minute YouTube narration.

💡 Free tier is enough — one chapter = one API call per day. You're well within limits.
POST https://generativelanguage.googleapis.com/
  v1beta/models/gemini-1.5-flash:generateContent
🎙️ GOOGLE TEXT-TO-SPEECH API · FREE TIER Neural2-D voice

Converts the Gemini script to audio using Google Cloud TTS. Uses the en-US-Neural2-D male voice — deep and dramatic, perfect for ORV.

Free tier: 1M characters/month for Neural voices. A 950-word script ≈ 5,500 chars. You'd need to process 181 chapters to hit the limit.

Voice: en-US-Neural2-D
SpeakingRate: 1.05
Pitch: -1.5
Returns: base64 MP3 audio
🎬 RENDER VIDEO (PYTHON) LOCAL · moviepy Panel slideshow + Ken Burns

The render_video.py script runs on your VM. It:

  • Loads all panel images + narration MP3
  • Processes each panel: blurred BG, gold border, watermark
  • Calculates duration per panel to match audio length
  • Applies Ken Burns zoom effect on alternate panels
  • Adds animated intro title card + subscribe outro
  • Exports 1920×1080 MP4 @ 4000kbps
⏱ Render time on Oracle VM: ~10–20 minutes per video. The workflow runs at 8AM, so video is ready well before prime viewing hours.
📺 UPLOAD TO YOUTUBE API · FREE TIER Data API v3 · OAuth2

The upload_youtube.py script uploads the rendered MP4 using resumable upload chunks (50MB each). Handles retries on server errors with exponential backoff.

Auto-generates: title (rotates 3 styles), description with hashtags, and a full tag list for discoverability.

📊 YouTube API quota: 10,000 units/day. One video upload = 1,600 units. You can upload up to 6 videos/day on the free tier.
✅ SAVE STATE + CLEANUP LOCAL Advance chapter counter

Updates orv_state.json to record the successfully uploaded chapter number. Then deletes the temporary panel images and audio file to free disk space.

The VM's 47GB free storage can handle ~3 chapters at once, but cleanup keeps it tidy.

2

Install n8n on the VM

ssh ubuntu@YOUR_VM_IP

# Install Node.js 20
curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
sudo apt-get install -y nodejs

# Install n8n globally
sudo npm install -g n8n

# Install PM2 to keep n8n running
sudo npm install -g pm2

# Start n8n as a background service
pm2 start n8n --name "n8n"
pm2 save
pm2 startup

n8n will be available at http://YOUR_VM_IP:5678 — open that port in Oracle's security rules.

3

Install Python Dependencies

sudo apt-get install -y python3-pip ffmpeg wget

# Install Python libraries
pip3 install moviepy Pillow numpy \
  google-auth google-auth-oauthlib \
  google-api-python-client

# Copy scripts to VM
mkdir -p /home/n8n
# (Upload render_video.py and upload_youtube.py here)

FFmpeg is required by moviepy for video encoding. It's free and available in Ubuntu's package manager.

4

Set Up Google Cloud APIs

Go to console.cloud.google.com → Create a new project called "ORV Automation".

Enable these 3 APIs:

1. Generative Language API (Gemini)
2. Cloud Text-to-Speech API
3. YouTube Data API v3

Go to APIs & Services → Credentials:

  • Create an API Key → use for Gemini + TTS in n8n
  • Create an OAuth 2.0 Client ID (Desktop App) → download JSON → save as /home/n8n/youtube_client_secrets.json
5

Authorize YouTube (One-Time)

python3 /home/n8n/upload_youtube.py \
  --video /tmp/test.mp4 \
  --title "Test" \
  --description "Test" \
  --tags "test" \
  --privacy private

This opens an OAuth URL in your terminal. Paste it in your browser, authorize with your YouTube account, paste back the code. A youtube_token.json is saved and auto-refreshed forever after.

💡 You only do this ONCE. After that, the workflow runs fully automated.
6

Import n8n Workflow

In n8n (http://VM_IP:5678):

  • Click + → Import from File
  • Upload orv_n8n_workflow.json
  • Add credentials: Google API Key, YouTube OAuth
  • Test run once manually to verify each node
  • Activate the workflow (toggle in top-right)

🎉 Done! The workflow will now run every morning at 8AM automatically.
